# Quick Start - Dynamic Interview Features

## 🚀 Start Using New Features in 3 Steps

### Step 1: Choose Your Duration (Interview Setup)

When setting up an interview, you'll see 5 duration options:

```
┌─────────────────────────────────────┐
│  5 minutes  →  15 questions         │
│ 10 minutes  →  20 questions         │
│ 15 minutes  →  20 questions  ⭐     │
│ 25 minutes  →  30 questions         │
│ 40 minutes  →  40 questions         │
└─────────────────────────────────────┘
```

**Recommendation**: Start with 15 minutes (20 questions) for a balanced experience.

---

### Step 2: Answer with Voice or Text

During the interview, you have two options:

#### Option A: Voice Input (Hands-Free) 🎤

1. Start speaking your answer
2. You'll see: **"🎤 Listening to the answer..."**
3. Stop speaking
4. Wait 2 seconds → Answer auto-submits!

**No "Submit" button needed!**

#### Option B: Text Input (Traditional) ⌨️

1. Type your answer in the text box
2. Click "Submit Answer" button
3. Done!

**You can mix both methods in the same interview!**

---

### Step 3: Complete the Interview

The interview automatically ends when you reach the question limit:

```
Question 20 of 20 → Submit answer
         ↓
AI evaluates your answer
         ↓
"That's all for this session. Thank you for attending.
 You will get to know your feedback very soon."
         ↓
Wait 3 seconds
         ↓
Auto-redirect to Results Page ✓
```

**No manual ending needed!**

## 💡 Pro Tips

### Voice Input Tips

✅ **DO:**
- Speak clearly at normal pace
- Wait for "Listening..." indicator
- Stop speaking completely before 2-second timer
- Use in quiet environment

❌ **DON'T:**
- Rush through your answer
- Speak while indicator is not showing
- Worry about pauses (timer resets)

### Duration Selection Tips

- **5 minutes (15 questions)**: Quick practice, specific topic
- **15 minutes (20 questions)**: Standard interview, balanced
- **25 minutes (30 questions)**: Comprehensive coverage
- **40 minutes (40 questions)**: Deep dive, full assessment

### Progress Tracking

Watch these indicators:
- **Progress Bar**: Visual representation of completion
- **Question Counter**: "Question 5 of 20"
- **Session Timer**: Total recording time

## 🔧 Troubleshooting

### Voice Input Not Working?

**Check:**
1. Microphone permissions allowed?
2. Using Chrome/Edge browser?
3. Microphone working in other apps?

**Fix:**
- Refresh page and allow permissions
- Try different browser
- Check system microphone settings

### Auto-Submit Not Triggering?

**Check:**
1. Did you stop speaking completely?
2. Waited full 2 seconds?
3. "Listening..." indicator showing?

**Fix:**
- Ensure complete silence for 2 seconds
- Speak louder/clearer
- Use manual "Submit Answer" button as backup

### Session Not Ending?

**Check:**
1. Reached question limit? (e.g., 15 of 15)
2. Last answer submitted?
3. Waited for evaluation?

**Fix:**
- Check progress bar for current question
- Wait for AI evaluation to complete
- Refresh if stuck (session will be saved)
